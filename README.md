# Dev Burguer

Página web para o Food Truck "Dev Burguer" - desenvolvida como atividade prática de HTML5 semântico.

## 📋 Sobre o Projeto

Um Food Truck de hambúrgueres artesanais com nomes de programadores (ex: "Hamburgão em C++", "Class Burger", "React Veggie").

**Objetivo:** Criar uma página web funcional com estrutura semântica HTML5, preparada para estilização com CSS no futuro.

---

## 📑 Conteúdo da Página

### Seções Obrigatórias

- **Cabeçalho (Header)**
  - Nome: "Dev Burguer"
  - Slogan: "O código mais saboroso da cidade"

- **Menu de Navegação**
  - Links para Cardápio
  - Links para Horário de Funcionamento

- **Cardápio**
  - Listagem dos 2 hambúrgueres mais vendidos
  - Cada item com: nome (negrito), descrição e preço

- **Horário de Funcionamento**
  - "Terça a Domingo, das 18h às 23h"

- **Rodapé (Footer)**
  - Copyright: "© 2025 Dev Burguer - Sabor em código"

### Seções Opcionais

- **Localização**
  - Endereço: "Rua das Linguagens, 404 - Centro"

---

## 🛠️ Requisitos Técnicos

### Tags HTML5 Semânticas
- `<header>` - Cabeçalho da página
- `<nav>` - Menu de navegação
- `<main>` - Conteúdo principal
- `<section>` - Seções de conteúdo
- `<footer>` - Rodapé da página

### Estrutura de Títulos
- `<h1>` - Título principal da página
- `<h2>` - Títulos das seções (Cardápio, Horário)

### Listas
- `<ul>` e `<li>` - Para os itens do cardápio

### Navegação
- Uso de atributo `id` nos elementos alvo
- Links âncora com `href="#id"`
- Navegação funcional ao clicar nos links

## 📝 Instruções de Desenvolvimento

### 1. Esboço
Desenhe no papel a estrutura (header, nav, main, footer) e identifique qual tag HTML5 será usada em cada bloco.

### 2. Pesquisa
Consulte a documentação se necessário:
- [W3Schools HTML Reference](https://www.w3schools.com/html/html_intro.asp)
- Livro base adotado: HTML5

### 3. Implementação
Crie o arquivo `index.html` com o código correspondente.

### 4. Validação
Teste no navegador:
- ✓ Os links do menu funcionam?
- ✓ A página rola até a seção correta ao clicar?
